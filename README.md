# uni-app 实战-前端全流程开发新闻资讯类应用
- 主分支为项目的完整代码。可以直接克隆到本地，本项目使用hbuilderx开发，配置好自己的云开发环境（关联云服务空间，上传云函数和仓库），预览完整效果。
- 支持H5、微信小程序、支付宝小程序，预览小程序端需百度官网下载小程序开发者工具（除微信小程序外均需运行hbuilderx后在相应小程序中配置路经）。
- 按F12 切换移动端预览
